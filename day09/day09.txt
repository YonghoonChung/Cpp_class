모든 클래스에는 객체를 복사하기 위해서 사용되는 복사생성자를 포함한다.

Object o
    t = o;
Object o(1)
    t(1) = o(1);

클래스 : 오버로딩

    오버로딩 : 같은 이름의 메소드/생성자 + 다른 매개변수값

    클래스에서 몇개의 오버로딩된 생성자 + 소멸자는 오직 한개

    부수적인 일 : 모든 클래스에는 동일한 다른 클래스의 다른 객체의 데이터로 초기화된 객체를 생성할 수 있다.
                (복사 생성자가 포함되어 있다.)

복사 생성자 생성
    ClassName(const ClassName&);

    Circle(const Circle& obj); // 복사 생성자에 의해서 만들어진 객체

    클래스에는 생성자, 복사 생성자, 소멸자가 기본적으로 

복사생성자와 대입연산자의 공통점/차이점
    공통점 : 하나의 객체로부터 다른 객체로 값을 대입한다.
    차이점 : 복사 -> 새로운 객체 생성, 대입 -> 새로운 객체를 생성 x

복사 생성자의 방법론: 
    1. 얕은 복사(=)
    2. 깊은 복사(포인터를 사용)

복사생성자 특징
    1. 복사 생성자의 명시적 타입 변환
    2. 복시 생성자의 처리영역
    3. 복사 생성자의 문자열
